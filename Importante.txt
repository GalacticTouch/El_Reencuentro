# 1. Revisa qué archivos cambiaste (opcional, pero recomendado para verificar)
git status

# 2. Prepara el archivo modificado para el envío (el punto agrega TODOS los cambios)
git add .

# 3. Ponle un nombre o descripción a tus cambios
git commit -m "Actualización de archivos"

# 4. Envía los cambios a tu repositorio en GitHub
git push origin main
